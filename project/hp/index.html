Review

  • CSS 인생은 전처리기를 도입하기 전과 후로 나뉜다

    한참 공부를 하던 중 한 블로그에서 본 글인데, SCSS를 공부하고 공감을 했습니다. 점점 방대해지는 CSS를 조금 더 가독성이 좋고 효율적으로 작성하고 싶었고, 이번 프로젝트에 SCSS를 도입하여 작업을 해보았습니다.

  • 인터랙티브 웹을 위한 첫걸음

    학원 선생님께서 이번에는 화려한 움직임의 작업을 권유해주셔서 작업을 하였고, GSAP ScrollTrigger Plugin을 이용하여 작업을 하였습니다. 처음 사용하다 보니 코드를 깔끔하고 정리되게 사용하지 못한것 같아 아쉬웠지만, 더욱 공부하고 싶어지는 분야가 되었습니다.

  • Mobile과 PC의 세계는 다르다

    반응형 작업을 위해 Plugin을 선택하여 작업을 한 것 이었는데, 처음 사용하다보니 디테일한 부분까지 터득하지 못하여 브라우저를 조절하면 레이아웃이 약간 틀어지는 현상있었고, 이 문제를 해결하기 위하여 'resize'이벤트를 이용하여 새로고침이 되도록 하였습니다.

  • 나도 할 수 있다! 가로스크롤

    화려한 움직임을 만들어내는 사이트들을 보면 가로스크롤을 이용한 페이지가 많아보였고, 바로 적용을 시켜보았습니다. position:sticky와 함께 간단하게 JavaScript 코드를 이용하여 작업을 하였습니다. 다만 모바일 환경에서 가로보다는 세로가 좋은거 같아 수정이 필요해 보입니다.
