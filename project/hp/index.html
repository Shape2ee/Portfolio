Review

  • React의 세계로 드디어 입장

    이번 프로젝트는 오로지 React만을 이용하여 구성하였습니다. 처음 작업하다 보니 궁금한 점도 많고, 학원에서 배운 것은 리액트의 일부여서 CSS와 자바스크립트로 쉽게 구현하던 것들을, React에서는 어떤 식으로 해야 할지 몰라 막막함도 있었지만, 밤낮을 가리지 않고 따로 틈틈이 강의 시청과 검색을 통해 반응형과 서브페이지까지 사이트를 구현하였습니다.

  • 하얀 도화지 같던 build 후 페이지...

    npm run build를 통해 build를 하고 index.html 파일을 열면 아무것도 없는 하얀색으로 페이지가 나왔고, 이유가 build 후 경로의 문제라는 것을 알게 되어 경로를 수정하였지만, src 폴더에 넣어둔 이미지들은 보이지 않는 문제가 있었습니다. 그래서 build 한 파일을 바로 github 페이지로 deploy 하는 것으로 문제를 해결하였습니다.

  • API 연동은 조금 더 공부하는 거로...

    API를 이용한 기능을 넣어보라는 선생님의 권유로 Pixabay API를 이용하여 검색한 이미지를 가져오는 기능을 추가하고, 구현이 잘 되었지만, 아직 Axios의 정확한 사용법을 이해하지 못한 것 같아서 다음 프로젝트에서 다시 사용하면서 공부할 예정입니다

  • 메인페이지가 보이지 않는 문제

    React Router를 이용하여 SPA인 React의 페이지 이동을 구현하였는데, 배포한 페이지의 path="/"를 적어준 메인화면이 보이지 않는 문제가 있었고 콘솔창을 확인해보니 경로를 찾지 못하는 문제여서 github repositories 이름인 "/hp-app-react"로 수정하여 문제를 해결하였습니다.
