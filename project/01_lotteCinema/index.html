overview

롯데시네마는 ‘HAPPY MEMORIES’ 라는 슬로건을 가지고 있는, 롯데컬처웍스가 운영하는 멀티플렉스 영화관 브랜드입니다.

기존 롯데시네마가 다른 영화관 브랜드 사이트보다 올드한 느낌이 강하다고 느껴져서 리디자인하였습니다. 롯데시네마가 주장하는 '백화점 속 영화관'이라는 기본 전략을 사이트에서도 느낄 수 있도록 하려고 영화관이지만 좀 더 고급지고 깔끔한 느낌을 중점으로 작업하였습니다.

Review

  • BEM 도전! 아쉬움이 남는 결과

    BEM 방식에 도전한다는 생각으로 작업을 했으나, 첫 시도이다보니 헷갈리는 부분이 있었고, 쉽게 생각했지만 신경쓸게 많아 공부를 더 해야할 것 같습니다.

  • 기다려라, 터치 & 드래그 이벤트

    처음 작업은 플러그인의 도움없이 지금까지 배운것만을 이용해서 작업을 해보고 싶어서 슬라이드를 작업을 도전했지만 드래그와 터치로는 작동을 시키지 못하여 아쉬운 부분입니다. 추후 공부해서 등록할 예정입니다.

  • Dropdown Custom

    브라우저 기본 스타일의 메뉴 말고 스타일을 만들려고 JavaScript로 작업을 하였는데, 메뉴들이 클릭 되지 않는 문제가 있었고,해결하려고 여러시도를 했지만 해결을 못하였습니다. 그러다 마지막으로 전체 코드를 보던 중 menu에 overflow:hidden 때문이라는 것을 발견하여 해당 코드를 지우는 것으로 문제를 해결하였습니다.

  • 비슷한 듯 다른 단위의 세계

    예매하기에서 “.tiketing-container”의 높이를 일정 사이즈일 때는 화면에 헤더 영역과 예매하기가 가득 보이게 만드려고 헤더를 뺀 100vh를 이용해서 만들고grid-template-rows를 1fr과 select존의 높이로 했는데 레이아웃이 계속 틀어졌고, 결국 1fr 아닌 %로 변경해서 작업하면서 문제를 해결했습니다.
