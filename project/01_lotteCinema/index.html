overview

롯데시네마는 ‘HAPPY MEMORIES’ 라는 슬로건을 가지고 있는, 롯데컬처웍스가 운영하는 멀티플렉스 영화관 브랜드입니다.

기존 롯데시네마가 다른 영화관 브랜드 사이트보다 올드한 느낌이 강하다고 느껴져서 리디자인하였습니다. 롯데시네마가 주장하는 '백화점 속 영화관'이라는 기본 전략을 사이트에서도 느낄 수 있도록 하려고 영화관이지만 좀 더 고급지고 깔끔한 느낌을 중점으로 작업하였습니다.

Review

  • BEM 도전! 아쉬움이 남는 결과

    BEM방식의 네이밍 방식을 적용하겠다는 생각을 가지고 작업하였고, 적용은 시켰지만 cSS에서의 사용이나 요소의 깊이 들어갔을때에 있어 완벽하게 하지는 못한거 같아, 개인적으로는 아쉬움이 남기도 합니다. 쉽게 생각했지만 신경쓸게 많아 공부를 더 해야할 것 같습니다.

  • 기다려라, 터치와 드래그

    처음 작업한 사이트이다 보니, 지금까지 배운 html,css,javaScript만을 이용해서 플러그인이 없이 작업을 해보고 싶어서 배너 슬라이드를 오로지 javascript로 작업해보았습니다. 다만 아직 마우스 드래그와 터치로는 작동이 되도록 하지 못하여 아쉬운 부분입니다. 추후 공부해서 등록할 예정입니다.

  • Dropdown Custom

    브라우저 기본 스타일의 dropdown 메뉴 말고 직접 만들고 싶어서 스타일을 만들고 javascript로 작업을 하였는데, 메뉴들이 클릭이 되지 않는 문제가 발생했고, blur 이벤트에 대한 이해가 부족하다고 생각하여 서치를 통해 찾아보았지만 해결하지 못하였고, 다시 처음으로 돌아가 전체 코드를 분석해 보았더니 menu에 overflow:hidden 때문이었다는 것을 발견하여 해당 코드를 지우는 것으로 문제를 해결하였습니다.

  • 단위마다 다른 레이아웃으로 인한 문제

    예매하기 페이지에서 “.tiketing-container”의 높이를 tablet 이상의 사이즈일 때는 화면에 헤더 영역과 예매하기가 가득 보이게 만드려고 100vh를 이용해서 헤더 영역을 빼주고 grid-template-rows를 이용해서 1fr과 select존의 높이로 주려고 했는데 레이아웃이 계속 틀어졌지만, 결국 1fr아닌 %로 변경해서 작업하면서 문제를 해결했습니다.
